## Some inexpensive starter boards:

Altera:

Terasic DE-10 Lite
$140 / $82 academic
Least number of issues with ports and drivers
http://de10-lite.terasic.com

Saylinx Bundle 2
$56
Well-rounded budget board with 8-digit 7-segment display and VGA
https://www.aliexpress.us/item/2255800651323111.html

Xilinx:

Digilent Basys 3 with Artix-7
$160
A convenient Xilinx board to start.
It is supported by Xilinx Vivado and therefore can be used with SystemVerilog synthesis,
unlike less expensive Spartan-6 boards, which can only be used with older Xilinx ISE toolchain.
https://digilent.com/shop/basys-3-artix-7-fpga-trainer-board-recommended-for-introductory-users

Gowin:

Tang Primer 20K Doc
$41
With a nice shield to accomodate the peripherals
https://www.aliexpress.us/item/3256804466994057.html
